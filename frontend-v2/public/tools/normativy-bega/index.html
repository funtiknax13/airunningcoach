Что такое ЕВСК и как присваиваются разряды

ЕВСК (Единая всероссийская спортивная классификация) — официальный документ, по которому в России присваиваются спортивные звания и разряды во всех видах спорта. Для присвоения разряда результат обычно должен быть показан на официальных соревнованиях, включённых в календарный план, и запротоколирован судейской коллегией — просто пробежать дистанцию в тренировочном темпе с личным GPS-треком для присвоения разряда недостаточно.

Разряды идут по возрастанию: III → II → I юношеский, затем III → II → I взрослый разряд, далее КМС (кандидат в мастера спорта) → МС (мастер спорта) → МСМК (мастер спорта международного класса).
